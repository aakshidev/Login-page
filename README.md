# Login-page

A clean and modern **Login Page UI** built using HTML and CSS with a premium **burgundy & white theme**. Designed to look minimal, elegant, and production-ready.

---

## ✨ Features

- 🎨 Burgundy + White color palette (premium feel)
- 📱 Fully responsive layout
- ⚡ Smooth entry animation
- 🧼 Clean and minimal UI design
- 🔒 Login form with username & password fields
- 🔗 "Forgot Password" link
- 🎯 Focus states and hover effects

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Variables, Animations)
